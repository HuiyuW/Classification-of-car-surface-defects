## Usage Description
* Clone or download three py files.  
* Add files to your datasets folder. Same path as annotated_functional_test3_fixed.json.  
* Change AL Epoch range (max_queried) as u want. Change models and selection_functions or Epoch size.  
```
max_queried = 200

repeats = 1

models = [SvmModel, RfModel, LogModel]

selection_functions = [MarginSamplingSelection, EntropySelection, MinStdSelection]

Ks = [50,25,15,10]
```  
* Run function AL.py  
* Process will be saved in pkl automatically.
* Enjoy and have fun!
