Manual Correction

WEBSITE FUNCTION INTRODUCTION

We have build the classification model for distinguishing the cropped damage patches in the four classes: "dent", "scratch", "rim" and "other". In this page, we will introduce how you could achieve the prediction of image classification, model refinement and dataset expansion through this website quickly and easily.

GET STARTED

Image Classification

Step1

Go to "Home" page and click "classification" button. Then click "Classification" on that page.

Step2: Upload images and choose model

Click "Select Images" to upload the images that you want to classify (You can choose multiple images at the same time and the supported formats of images are .jpg, .jpeg, .webp, .png). After choosing images, make sure to submit your choices by clicking "submit" button. Then you can see your chosen images on the screen. Choose a model in "Model choosing" menu (the default option is InceptionV3 model). If you choose InceptionV3 model or ResNet18 model. Just confirm your chosen model by clicking "confirm" button and you can directly click "Detection" and go to "Final classification report and Download" module in this site to see how to check the results and get the final report.

Active Learning and Model Refine

1. If you choose Active Learning model, system will automatically choose "ResNet18 + Active learning" for classification. You can choose how many uncertainty images you want the model to choose for you and choose which strategy you decide to use in active learning. Then click "confirm" button to save your choices and click "Detect" button.

2. Next, "active learning" method will choose the uncertainty images for you to label. If you think the labels of the images are already correct, click "Add to model". If you think the label of the corresponding image is not correct, just enable the "Manual Marketing" button and choose the correct label that you think and click "change" button. Then the label of this image will be changed and don't forget to add it to model by clicking "Add to Model".(Dont't forget to click "add to model" for all images in this website page!)

3. You can see the information of all images which was chosen by active learning with correct labels in page "Active Learning". And you can view these images through "view" button. By clicking "Delete" button, you can delete the corresponding images if you don't want to use it for refining the model. Click the button "Refine Model" to use the images in this page to refine our machine learning model. This process will take a few minutes. After refining, you can see the active learning model analysis report in this page. The active learning and model refine part ends here and you can go to "Final classification report and Download" module in this website to get the finally report.

Final classification report and Download

Finally let's go to Download page. Information of images which were selected and uploaded are shown here no matter which model you have chosen for classification. You can also view the images through "view" button and delete images through "Delete" button. If you think the label of the corresponding image is wrong, just choose the correct label of this image and click "change label" button. Then the label will be changed. Finally you can get the final classification report as json file by clicking the "Download" button. Please click "clear" on this page to clear the images in this page if you want to reselect images for a new round of classification.




Data Expansion

You can expand the dataset through "Image Label" page in this website.
After uploading the unlabeled images, these images will show in the "Unlabeled Image table". Then you can choose the label for the corresponding image in the "Label-choise" column in "Unlabeled Image Table". The unlabeled images will go into the "Labeled Image Table" after labeling. If you accidentally mislabeled the image, you can remove the image by clicking "remove" in the "Labeled Image table" and the image will return to "Unlabeled Image List". After labeling all images, the information of labeled images can be downloaded as Json file with name "image_label.json". "Labeled Images" and "Unlabeled Images" are buttons that help you quickly go to the corresponding list.
